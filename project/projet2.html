This project is about designing and implementing a smart home system using IoT technology. It allows users to control devices like lights, fans, and security systems remotely.

Challenges and Learnings

One major challenge was ensuring reliable communication between devices. I learned a lot about debugging IoT protocols and optimizing network efficiency.
